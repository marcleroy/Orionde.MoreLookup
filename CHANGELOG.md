## [0.9.0-beta] - 2025-11-18

### Added
- Initial beta release
- Union operation for combining lookups
- Intersect operation for finding common elements
- Except operation for set difference
- Join operation for joining two lookups
- Zip operation for combining corresponding elements
- Select operation for value transformation
- Where operation for filtering values
- OnEachKey operation for key-specific transformations
- Concat operation for concatenating lookups
- Fluent builder pattern (Lookup.Builder)
- Support for custom key and value comparers
- Empty lookup singleton (Lookup.Empty<TKey, TValue>())
- Dictionary to Lookup conversions
- Lookup to Dictionary conversions
- Multi-framework support (.NET Framework 4.8, .NET 6, 7, 8, 9, .NET Standard 2.0)
- Comprehensive XML documentation
- Full test coverage

### Documentation
- Complete README with examples
- API documentation via XML comments
- Comparison with other libraries (MoreLinq)
- Acknowledgment of inspiration from NOtherLookup
